Java 5DayTallinnForecast

An android app that gets the weather forecast for the next 5 days and nights for Tallinn Estonia. The data is pulled from http://openweathermap.org/api and extracted from a JSON response. There's an unique icon that's tied to each weather condition and is displayed accordingly. The program uses a custom-built ArrayAdapter to inflate the ListView with Weather objects. Each Weather object consists of a weather condition, a weather icon code, date and time of the weather, temperature and wind speed. The program was written using Android Studio.
