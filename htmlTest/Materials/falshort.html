Falstad.com/circuit shortcuts

Ülevalt menüüst Circuits -> Blank Circuit

Shortcuts

  • N (shift + n) - nMOS transistor, right click -> edit -> uncheck Show Bulk
  • P (shift + p) - nMOS transistor
  • w - juhe
  • V (shift + v) - +5V
  • g - maandus/ground
  • i - loogikasisend/input
  • o - loogikaväljund/output
  • t - text (cMOS AND/OR, nimi, rühm)
  • space/tühik - elementide liigutamine

Et sisend ja väljund juhtme keskele saada, tuleb teha mitu lühikesemat juhet, siis saab juhtme ühenduskohta need külge keevitada
Ülevalt menüüst - File -> export as link - kui on valmis.
